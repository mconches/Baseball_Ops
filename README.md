# Baseball Operations Analytics Pipeline

A professional-grade data pipeline built in **Python** and **R** for extracting and visualizing MLB Statcast data.

## Features
- **Data Acquisition**: Automated scraping of Baseball Savant and FanGraphs via `pybaseball` and `baseballr`.
- **Pitch Profiling**: Detailed movement analysis (Ride/Sink) for MLB pitchers.
- **Cross-Language Validation**: Dual-pipeline approach using Python (Pandas/Seaborn) and R (Tidyverse/ggplot2) to ensure data integrity.

## Tech Stack
- **Python**: Pandas, Matplotlib, Seaborn
- **R**: Tidyverse, baseballr
- **Environment**: VS Code & RStudio
